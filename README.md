# 用户管理系统（原始版本 - 含安全漏洞）

> ⚠️ **此分支包含大量的安全漏洞，仅用于安全学习和漏洞对比**
> 
> 安全加固版本请查看 `main` 分支

## ⚠️ 已知安全漏洞

此版本**故意保留**了以下安全漏洞，用于教学演示：

| 严重等级 | 漏洞 | 说明 |
|---------|------|------|
| 🔴 严重 | 明文密码存储 | 密码以明文存储在代码中 |
| 🔴 严重 | Secret Key 硬编码 | `dev-key-2025` 可伪造 session |
| 🔴 严重 | Debug 模式开启 | 可远程执行代码 |
| 🟠 高危 | 无 CSRF 保护 | 可跨站请求伪造 |
| 🟠 高危 | Cookie 无安全属性 | 无 HttpOnly 和 SameSite |
| 🟠 高危 | 密码展示在前端 | 页面直接显示密码 |
| 🟡 中危 | HTML 泄露账号 | 源码注释泄露管理员密码 |
| 🟡 中危 | 无暴力破解限制 | 可无限次尝试密码 |
| 🟡 中危 | 无 HTTPS | 密码明文传输 |
| 🟢 低危 | 无输入校验 | 可能导致 DoS |

## 快速启动

```bash
pip install flask
python3 app.py
```

访问 http://127.0.0.1:5000

默认账号：admin / admin123
